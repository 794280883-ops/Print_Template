## [1.0] — 首个完整功能版本

### 模板管理
- 模板列表（CRUD + 批量启用/停用 + 复制 + 搜索）
- 可视化模板设计器（拖拽组件：文本/横线/矩形/一维码/二维码/复选框）
- 模板属性编辑、字段绑定、撤销重做、复制粘贴
- 模板启用/停用校验（未绑定字段/元素超出画布等检查）

### 字段字典
- 树形分类（库位/容器/商品 + 自定义模块）
- 字段 CRUD + 启用/停用
- 启用查询开关（控制业务数据搜索区字段）

### 业务数据管理
- 统一 `business_record` 表存储（JSON字段）
- 查询（编码批量查询 + 字段精确筛选）
- CRUD + 批量删除 + Excel 导入/导出模板

### 打印功能
- 选择模板 + 数据 → 生成 PDF 下载
- 条码（bwip-js）+ 二维码（qrcode）生成
- PDFKit 中文支持（ArialUnicode 字体）
- 打印日志记录
- 记忆用户上次使用的打印模板

### 权限系统（RBAC）
- 五表模型：sys_user / sys_role / sys_menu / sys_user_role / sys_role_menu
- JWT 登录（24h过期）
- 前端：路由守卫 + v-permission 指令
- 后端：requirePermission 中间件
- 菜单管理（目录/页面/按钮三级）
- 角色权限卡片式分配
- 空目录自动隐藏

### 系统预置
- 3个业务模块：库位(LOCATION) / 容器(CONTAINER) / 商品(PRODUCT)
- 3个预置模板：库位标签-100x50 / 容器入库标签-100x50 / 拣货容器标签-100x150
- 2个预置账号：admin / business
- 14个库位字段 + 3个容器字段 + 3个商品字段

### 部署
- Docker Compose 编排（MySQL + Backend + Frontend + Nginx）
- Nginx HTTPS 反向代理
- Let's Encrypt 证书自动续期
- 阿里云 ECS 一键部署脚本
- 预提交检查脚本
