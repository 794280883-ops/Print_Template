# 接口文档

本文档列出打印模板中心已实现的后端 API 接口。

## 基础信息

Base URL：

```text
/api/v1
```

统一响应格式：

```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

错误码：

```text
40000 参数错误
40001 模板编码重复
40003 模板名称重复
40400 数据不存在
40900 业务编码冲突
50000 系统异常
```

## 模板管理

```text
GET    /templates                    获取模板列表
POST   /templates                    创建模板
GET    /templates/:id                获取模板详情
PUT    /templates/:id                更新模板（全量）
PATCH  /templates/:id/name           更新模板名称
POST   /templates/:id/design-log     记录设计日志
POST   /templates/:id/enable         启用模板
POST   /templates/:id/disable        停用模板
POST   /templates/:id/copy           复制模板
DELETE /templates/:id                删除模板
POST   /templates/status             批量更新模板状态
```

## 字段字典

```text
GET    /template/fields/:templateType    获取字段字典
```

模板类型（大写）：

- `LOCATION` — 库位字段
- `CONTAINER` — 容器字段
- `PRODUCT` — 商品字段

## 业务数据

```text
GET    /business-data/types                              获取业务类型
GET    /business-data/warehouses                         获取仓库列表
GET    /business-data/search                             搜索业务数据
GET    /business-data/detail/:bizType/:bizCode           获取业务数据详情
POST   /business-data                                    新增业务数据
PUT    /business-data/:bizType/:bizCode                  更新业务数据
DELETE /business-data/:bizType/:bizCode                  删除业务数据
GET    /business-data/template/:bizType                  下载导入模版
POST   /business-data/import/:bizType                    导入业务数据
```

业务类型：

- `LOCATION` — 库位
- `CONTAINER` — 容器
- `PRODUCT` — 商品

## 打印

```text
POST   /print/pdf     生成并下载打印 PDF
```

## 操作日志

```text
GET    /operation-logs    获取操作日志
```

## 健康检查

```text
GET    /health    服务健康检查
```
