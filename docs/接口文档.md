# 接口文档

本文档列出打印模板中心已实现的后端 API 接口。

## 基础信息

Base URL：

```text
/api/v1
```

统一响应格式：

```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

错误码：

```text
40000 参数错误
40001 编码重复（模板/模块/字段）
40002 引用冲突 / DSL 校验失败 / 必填字段不允许删除
40003 模板名称重复
40004 模板未启用
40005 模板类型不匹配
40400 数据不存在
50000 系统异常
```

## 模板管理

```text
GET    /templates                    获取模板列表
POST   /templates                    创建模板
GET    /templates/:id                获取模板详情
PUT    /templates/:id                更新模板（全量）
PATCH  /templates/:id/name           更新模板名称
POST   /templates/:id/design-log     记录设计日志
POST   /templates/:id/enable         启用模板
POST   /templates/:id/disable        停用模板
POST   /templates/:id/copy           复制模板
DELETE /templates/:id                删除模板
POST   /templates/status             批量更新模板状态
```

## 字段字典

```text
GET    /template/fields/:moduleCode    获取字段字典
```

模块类型（大写）：

- `LOCATION` — 库位字段
- `CONTAINER` — 容器字段
- `PRODUCT` — 商品字段
- 自定义模块编码

## 业务模块

```text
GET    /business-modules                                  获取业务模块列表
POST   /business-modules                                  创建业务模块
PUT    /business-modules/:moduleCode                      更新业务模块
DELETE /business-modules/:moduleCode                      删除业务模块
POST   /business-modules/:templateType/fields             新增字段（⚠️ 参数名为 :templateType，不是 :moduleCode）
PUT    /business-modules/:templateType/fields/:fieldCode  更新字段
POST   /business-modules/:templateType/fields/:fieldCode/disable  停用字段
POST   /business-modules/:templateType/fields/:fieldCode/enable   启用字段
DELETE /business-modules/:templateType/fields/:fieldCode  删除字段
```

模块响应字段：

```json
{
  "code": "LOCATION",
  "name": "库位",
  "templateLabel": "库位模板",
  "dataLabel": "库位数据",
  "codeField": "locationCode",
  "recordCodeField": "locationCode",
  "storageMode": "json_table"
}
```

系统内置 LOCATION / CONTAINER / PRODUCT 不可删除。

## 业务数据

所有业务数据统一存储在 `business_record` 表，字段 schema 由运行时动态编译。

```text
GET    /business-data/types                              获取业务类型列表
GET    /business-data/search                             搜索业务数据
POST   /business-data                                    新增业务数据
PUT    /business-data/:bizType/:bizCode                  更新业务数据
DELETE /business-data/:bizType/:bizCode                  删除业务数据
DELETE /business-data/batch/:bizType                     批量删除业务数据
GET    /business-data/template/:bizType                  下载导入模版
POST   /business-data/import/:bizType                    导入业务数据
```

业务类型 `bizType` 即 `module_code`。

### 搜索参数

| 参数 | 说明 |
|------|------|
| `bizType` | 业务类型（LOCATION/CONTAINER/PRODUCT/自定义模块编码） |
| `keyword` | 业务编码关键字，支持逗号或空格分隔批量查询（模糊匹配 record_code） |
| `fieldFilters` | 按字段精确查询，JSON 对象格式，如 `{"warehouseCode":"DE","areaCode":"DE-1"}`，仅对启用了查询的字段生效 |
| `page` | 页码 |
| `pageSize` | 每页条数 |
| `sortField` | 排序字段 |
| `sortDir` | 排序方向（ASC/DESC） |

## 认证与授权

```text
POST   /auth/login            登录（返回 token + 权限码 + 菜单树）
GET    /users                 用户列表
POST   /users                 新增用户
PUT    /users/:id             编辑用户
DELETE /users/:id             删除用户
PUT    /users/:id/password    修改密码
GET    /roles                 角色列表
POST   /roles                 新增角色
PUT    /roles/:id             编辑角色
DELETE /roles/:id             删除角色
GET    /roles/:id/menus       查看角色菜单权限
PUT    /roles/:id/menus       分配角色菜单权限
GET    /menus                 菜单列表
POST   /menus                 新增菜单
PUT    /menus/:id             编辑菜单
DELETE /menus/:id             删除菜单
```

菜单类型: `directory`(目录) / `page`(页面) / `button`(按钮)。按钮级权限通过 `permission_code` 控制。

## 字段属性说明

字段字典（`GET /template/fields/:moduleCode`）返回的字段属性：

| 属性 | 说明 |
|------|------|
| `code` | 字段编码 |
| `name` | 字段中文名称 |
| `type` | 字段类型（string/number/integer/date/select） |
| `required` | 是否必填 |
| `searchable` | 是否启用查询（true=在业务数据中可按该字段精确查询） |
| `sortable` | 是否支持排序 |
| `enabled` | 是否启用 |
| `example` | 示例值 |
| `desc` | 字段说明 |
| `sortNo` | 排序号（必填） |

业务编码字段（主键）默认 `searchable=true` 且不可取消。

## 请求示例

**创建模板：**

```json
{
  "templateCode": "LOCATION_20260615001",
  "templateName": "库位标签-100x50",
  "templateType": "LOCATION",
  "size": { "width": 100, "height": 50, "unit": "mm", "dpi": 203 },
  "remark": "",
  "elements": []
}
```

**新增业务数据：**

```json
{
  "bizType": "LOCATION",
  "fields": {
    "locationCode": "A01-001",
    "warehouseCode": "JP-TYO-01"
  }
}
```
