# 用户操作手册

本文档以通俗易懂的方式说明打印模板中心各功能的操作步骤，帮助业务人员快速上手使用系统。

---

## 1. 登录与退出

### 1.1 登录系统

1. 打开浏览器，访问系统网址，进入登录页面
2. 输入分配给您的账号和密码
3. 点击「登录」按钮进入系统

**账号说明**：
- `admin` / `--` — 系统管理员，拥有全部功能权限
- `business` / `123456` — 业务操作员，可使用模板设计和业务数据功能

### 1.2 退出登录

1. 点击页面右上角的账号名称
2. 点击「退出登录」
3. 页面跳转回登录界面

## 2. 新建并设计打印模板

### 2.1 创建空白模板

1. 进入系统后，点击左侧菜单「模板列表」
2. 点击页面右上角的「新增模板」按钮
3. 在弹出窗口中填写模板信息：
   - **模板名称**：给模板起个名字，例如"库位标签-100x50"
   - **模板类型**：选择这个模板用于哪类业务（库位/容器/商品）
   - **宽度/高度**：设置打印纸的尺寸，单位是毫米
4. 点击「确定」按钮，自动进入模板设计界面

### 2.2 认识模板设计器

模板设计器是画打印标签的地方，界面分为三个区域：

- **左侧**：放组件的地方，有文字、二维码、条码等元素
- **中间**：画布，就是模板的实际样子
- **右侧**：选中某个元素后，可以修改它的属性

### 2.3 添加文字

**添加固定文字**（打印时内容不变）：
1. 在左侧「组件区」找到「静态文本」
2. 拖拽到中间画布上
3. 在右侧设置显示的文字内容、字体大小、颜色等

**添加动态文字**（打印时显示实际数据）：
1. 在左侧「模版字段」中找到需要的字段
2. 拖拽到中间画布上
3. 这个文字框会自动关联对应字段，打印时显示实际数据值

### 2.4 添加二维码

1. 在左侧「组件区」找到「二维码」
2. 拖拽到中间画布上
3. 在右侧「绑定字段」下拉框中选择要关联的字段
4. 调整二维码的大小（二维码需要足够的尺寸才能正常扫描）

### 2.5 添加条码

1. 在左侧「组件区」找到「一维码」
2. 拖拽到中间画布上
3. 在右侧「绑定字段」下拉框中选择要关联的字段
4. 如需显示人工可读的数字，勾选「显示文字」选项

### 2.6 添加装饰元素

| 元素 | 用途 |
|------|------|
| 横线 | 添加水平或垂直的分隔线 |
| 矩形 | 添加带背景色的方块，可用于背景底色 |
| 复选框 | 添加可勾选的方框 |

### 2.7 调整元素

- **移动位置**：用鼠标拖拽元素到目标位置
- **改变大小**：用鼠标拖拽元素边缘的蓝色方块
- **旋转方向**：在右侧属性中设置旋转角度（0°/90°/180°/270°）
- **删除元素**：选中元素后，按键盘上的 Delete 或 Backspace 键

### 2.8 复制粘贴元素

1. 选中要复制的元素
2. 按键盘 `Ctrl + C`（苹果电脑是 `Command + C`）
3. 按键盘 `Ctrl + V`（苹果电脑是 `Command + V`）

### 2.9 撤销和重做

- **撤销**：`Ctrl + Z`（苹果电脑是 `Command + Z`）— 取消上一步操作
- **重做**：`Ctrl + Y`（苹果电脑是 `Command + Y`）— 恢复被撤销的操作

### 2.10 保存模板

1. 点击页面右上角的「保存」按钮
2. 系统会检查模板是否正确：
   - 如果显示错误提示，根据提示修改后重新保存
   - 如果提示保存成功，表示模板已保存好

**常见问题**：
- 提示"画布内必须有元素"：需要先在画布上添加组件
- 提示"动态元素未绑定字段"：需要为文字/二维码/条码关联具体字段
- 提示"元素超出画布"：需要把元素拖回画布范围内

### 2.11 预览效果

1. 点击页面右上角的「预览」按钮
2. 在弹出窗口中查看模板打印出来的实际效果
3. 确认无误后关闭预览窗口

### 2.12 调整显示比例

在页面顶部的工具栏中：
- 选择缩放比例：75% / 100% / 125% / 150%
- 开启/关闭网格辅助线（开启后元素会自动对齐）

## 3. 管理已有模板

在「模板列表」页面操作。

### 3.1 修改模板

找到要修改的模板，点击操作列的「编辑」按钮，进入设计界面进行修改。

### 3.2 删除模板

1. 找到要删除的模板
2. 点击操作列的「删除」按钮
3. 在弹出确认框中点击「确定」

### 3.3 启用或停用模板

- **启用**：将停用的模板设为可用，只有可用的模板才能用于打印
- **停用**：将模板暂时设为不可用，不影响已打印的内容

### 3.4 复制模板

1. 找到要复制的模板
2. 点击操作列的「复制」按钮
3. 系统会创建一个内容完全相同的副本，名称后面加"副本"
4. 可进入副本的编辑界面进行修改

### 3.5 打印模板

1. 找到要使用的模板
2. 点击操作列的「打印」按钮
3. 在弹出窗口中勾选要打印的业务数据
4. 设置打印份数
5. 点击「确认打印」，系统生成 PDF 文件并下载

## 4. 管理字段

字段是打印模板会用到的数据项目，例如"库位编码"、"商品条码"。

路径：左侧菜单「模版字段」

### 4.1 查看字段

在顶部 Tab 栏切换不同的业务模块，查看各模块下的字段列表。

### 4.2 新建业务模块

1. 点击「新增模块」按钮
2. 填写模块信息：
   - **模块编码**：模块的唯一标识，如 `PALLET`（创建后不可修改）
   - **模块名称**：模块的中文名称，如"托盘"
   - **模板类型名称**：用于打印模板时显示，如"托盘模板"
   - **业务数据名称**：用于业务数据时显示，如"托盘数据"
   - **主键字段编码**：该模块数据的主键字段，如 `palletCode`
   - **主键字段名称**：主键字段的中文名称，如"托盘编码"
3. 点击「确定」

### 4.3 新增字段

1. 切换到目标模块的 Tab 页
2. 点击「新增字段」按钮
3. 填写字段信息：
   - **字段编码**：字段的唯一标识，如 `palletType`
   - **中文名称**：字段显示名称，如"托盘类型"
   - **字段类型**：选择数据类型（字符/数值/整数/日期）
   - **必填**：勾选后该字段必须填写才能保存
   - **启用查询**：勾选后该字段在业务数据中支持精确查询（默认勾选；业务编码字段强制勾选且不可取消）
   - **支持排序**：勾选后可在列表中按该字段排序
   - **排序号**：决定字段显示顺序（必填，新增时默认为空需手动填写）
4. 点击「确定」

### 4.4 修改字段

1. 在字段列表中，点击操作列的「编辑」按钮
2. 修改需要调整的内容（字段编码不可修改）
3. 点击「确定」

### 4.5 停用或启用字段

- **停用**：将字段暂时设为不可用状态，字段还在但暂时不使用
- **启用**：将停用的字段重新恢复可用

### 4.6 删除字段

1. 在字段列表中，点击操作列的「删除」按钮
2. 在弹出确认框中点击「确定」

**注意**：必填字段不允许删除。如需删除，请先编辑该字段取消必填，或停用该字段。

### 4.7 修改模块信息

1. 切换到目标模块的 Tab 页
2. 点击「编辑模块」按钮
3. 修改模块名称等信息（模块编码不可修改）
4. 点击「确定」

### 4.8 删除业务模块

1. 切换到目标模块的 Tab 页
2. 点击「删除模块」按钮
3. 在弹出确认框中点击「确定」

**注意**：系统预置的库位、容器、商品模块不可删除。

## 5. 管理业务数据

业务数据是实际要打印的内容，例如具体的库位"A01-001"、商品"SKU-12345"。

路径：左侧菜单「业务数据」

### 5.1 切换业务类型

在顶部 Tab 栏选择要管理的业务类型（库位/容器/商品）。

### 5.2 查询数据

业务数据页面支持两种查询方式：

**业务编码查询**：在"业务编码"输入框中输入编码关键字，支持用逗号或空格分隔输入多个编码进行批量查询。

**字段精确查询**：在模版字段中勾选了"启用查询"的字段，会自动在搜索区域显示对应的输入框。在输入框中输入值后点击「查询」，系统会按该字段精确匹配。多个字段可同时填写，系统按所有条件组合筛选。

点击「重置」按钮可清空所有查询条件。

### 5.3 新增数据

1. 点击「新增」按钮
2. 在弹出表单中填写各项数据（带 * 号的为必填项）
3. 点击「确定」保存

### 5.4 修改数据

1. 在数据列表中，点击要修改的那一行编码列
2. 在弹出表单中修改数据内容
3. 点击「确定」保存

### 5.5 删除数据

**删除单条**：在操作列点击「删除」按钮

**批量删除**：
1. 勾选要删除的多条数据（左侧复选框）
2. 点击「删除」按钮
3. 在弹出确认框中点击「确定」

### 5.6 导入数据（Excel 批量导入）

1. 点击「导入」按钮
2. 点击「下载导入模板」下载 Excel 文件
3. 在 Excel 中按照格式填写数据
4. 回到系统，点击上传区域或拖拽上传填写好的 Excel 文件
5. 点击「确认导入」
6. 查看导入结果，确认成功条数和失败原因

### 5.7 打印数据

1. 勾选要打印的数据（可多选）
2. 点击「打印」按钮
3. 在弹出窗口中选择要使用的模板（左侧列表选择，右侧预览效果）
4. 设置打印份数（1-99份）
5. 点击「确认打印」，系统生成 PDF 文件并下载

## 6. 系统管理

### 6.1 用户管理

路径：左侧菜单「系统管理」→「用户管理」

| 操作 | 步骤 |
|------|------|
| 新建用户 | 点击「新增」，填写账号、密码、姓名，选择岗位角色 |
| 修改用户 | 点击操作列「编辑」，修改用户信息 |
| 修改密码 | 点击操作列「修改密码」，输入新密码 |
| 停用/启用用户 | 点击操作列的停用/启用按钮 |
| 删除用户 | 点击操作列「删除」，在确认框中点击确定 |

### 6.2 角色管理

路径：左侧菜单「系统管理」→「角色管理」

角色是一组权限的集合，给用户分配角色就是分配权限。

| 操作 | 步骤 |
|------|------|
| 新建角色 | 点击「新增」，填写角色编码和名称 |
| 修改角色 | 点击操作列「编辑」，修改角色名称 |
| 删除角色 | 点击操作列「删除」（有用户关联的角色不能删） |
| 分配权限 | 点击操作列「编辑」，勾选允许使用的功能 |

**权限分配说明**：
- 勾选页面名称 = 给该岗位开放这个功能页面的访问权限
- 点击蓝色的权限标签 = 给该岗位开放这个按钮的操作权限
- 点击「全选」= 一次开放所有权限
- 点击「清空」= 一次取消所有权限

### 6.3 菜单管理

路径：左侧菜单「系统管理」→「菜单管理」

菜单是左侧的功能导航树，菜单管理用于维护这个导航结构。

| 操作 | 步骤 |
|------|------|
| 新建目录 | 点击「新增顶级目录」，填写目录名称 |
| 新增页面 | 选中一个目录后点击「新增」，选择类型为"页面" |
| 新增按钮 | 选中一个页面后点击「新增」，选择类型为"按钮" |
| 修改菜单 | 点击操作列「编辑」，修改菜单属性 |
| 删除菜单 | 点击操作列「删除」（有子菜单的不能删） |

## 7. 常见问题

### 7.1 保存模板时报错"元素未绑定字段"

动态文字、二维码、条码这些组件需要关联具体的字段才能正常工作。请在右侧属性面板中找到"绑定字段"下拉框，选择要关联的字段。

### 7.2 启用了模板但打印不了

请确认：
1. 模板状态是"启用"（绿色）
2. 打印时选择的业务数据类型与模板类型一致（如库位模板对应库位数据）

### 7.3 导入 Excel 失败

请检查：
1. Excel 文件格式是否为 .xlsx（不是 .xls）
2. 必填的字段是否都填写了
3. 主键字段是否有重复（重复的主键会导入失败）
4. 查看导入结果中的错误提示，根据提示修改后重新导入

### 7.4 打印出来的内容和预览不一样

请检查：
1. 预览时选择的模板是否和打印时一样
2. 模板中的动态字段是否都正确绑定了字段
3. 要打印的业务数据是否都已经填写了对应字段的值
