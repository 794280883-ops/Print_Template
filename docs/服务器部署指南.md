# 服务器部署指南

本文档说明如何将 打印模板中心部署到阿里云服务器。

## 服务器信息

```text
公网 IP：47.113.118.74
主私网 IP：172.16.117.184
部署目录：/opt/wms-print-template-center
```

## 1. 阿里云安全组

开放入方向：

```text
22/tcp   SSH 管理
80/tcp   HTTP 访问
443/tcp  后续 HTTPS
```

不要开放：

```text
3306/tcp MySQL
3001/tcp Backend
```

## 2. 服务器初始化

首次部署时需要安装 Docker：

```bash
sudo apt update
sudo apt install -y git ca-certificates curl gnupg

sudo install -m 0755 -d /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
sudo chmod a+r /etc/apt/keyrings/docker.gpg

echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu \
  $(. /etc/os-release && echo "$VERSION_CODENAME") stable" | \
  sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

sudo apt update
sudo apt install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
sudo systemctl enable --now docker
```

## 3. 拉取项目

```bash
sudo mkdir -p /opt/wms-print-template-center
sudo chown -R "$USER":"$USER" /opt/wms-print-template-center

git clone git@github.com:794280883-ops/Print_Template.git /opt/wms-print-template-center
cd /opt/wms-print-template-center
```

如果服务器没有配置 GitHub SSH Key，可临时使用 HTTPS：

```bash
git clone https://github.com/794280883-ops/Print_Template.git /opt/wms-print-template-center
```

## 4. 创建生产环境变量

```bash
cp .env.production.example .env.production
nano .env.production
```

至少替换：

```text
MYSQL_ROOT_PASSWORD=强密码
MYSQL_PASSWORD=强密码
CORS_ORIGIN=http://47.113.118.74
HTTP_PORT=80
```

## 5. 启动服务

```bash
docker compose --env-file .env.production up -d --build
```

查看状态：

```bash
docker compose --env-file .env.production ps
docker compose --env-file .env.production logs -f backend
```

## 7. 一键部署（日常更新）

本地验证并推送代码后，执行：

```bash
./scripts/deploy-server.sh
```

脚本默认配置：

```text
SERVER_HOST=47.113.118.74
SERVER_USER=root
SERVER_DIR=/opt/wms-print-template-center
SSH_KEY=/tmp/wms_deploy_key
BRANCH=main
```

如需指定 SSH Key：

```bash
SSH_KEY=~/.ssh/wms_deploy_key ./scripts/deploy-server.sh
```

## 8. 常用运维命令

**更新代码并重启：**

```bash
cd /opt/wms-print-template-center
git pull
docker compose --env-file .env.production up -d --build
```

**停止服务：**

```bash
docker compose --env-file .env.production down
```

**备份 MySQL：**

```bash
docker exec wms-print-template-mysql sh -c 'mysqldump -uroot -p"$MYSQL_ROOT_PASSWORD" wms_print_template' > wms_print_template_$(date +%Y%m%d_%H%M%S).sql
```

## 9. 域名说明

已配置域名解析：

```text
customprint.icu
www.customprint.icu
```

由于服务器在中国内地，域名正式访问需要完成 ICP 备案。备案完成前建议继续使用公网 IP。

## 10. 生产环境注意事项

- 替换所有 `.env` 中的敏感信息后再部署
- 如需 HTTPS，在 Nginx 前配置 TLS 或修改 `nginx/default.conf` 添加证书
- 数据库切换前执行 `npm run migrate`
- 公开写接口前需添加鉴权
- MySQL 备份需在应用容器外配置
