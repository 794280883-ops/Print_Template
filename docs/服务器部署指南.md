# 服务器部署指南

本文档说明如何将 打印模板中心部署到阿里云服务器。

## 服务器信息

```text
公网 IP：47.113.118.74
主私网 IP：172.16.117.184
部署目录：/opt/wms-print-template-center
```

## 1. 阿里云安全组

开放入方向：

```text
22/tcp   SSH 管理
80/tcp   HTTP（跳转 HTTPS）
443/tcp  HTTPS
```

不要开放：

```text
3306/tcp MySQL
3001/tcp Backend
```

## 2. 服务器初始化

首次部署时需要安装 Docker：

```bash
sudo apt update
sudo apt install -y git ca-certificates curl gnupg

sudo install -m 0755 -d /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
sudo chmod a+r /etc/apt/keyrings/docker.gpg

echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu \
  $(. /etc/os-release && echo "$VERSION_CODENAME") stable" | \
  sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

sudo apt update
sudo apt install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
sudo systemctl enable --now docker
```

## 3. 拉取项目

```bash
sudo mkdir -p /opt/wms-print-template-center
sudo chown -R "$USER":"$USER" /opt/wms-print-template-center
```

## 7. 一键部署（日常更新）

本地验证并推送代码后，执行：

```bash
./scripts/deploy-server.sh
```

脚本默认配置：

```text
SERVER_HOST=47.113.118.74
SERVER_USER=root
SERVER_DIR=/opt/wms-print-template-center
SSH_KEY=/tmp/wms_deploy_key
BRANCH=main
```

如需指定 SSH Key：

```bash
SSH_KEY=~/.ssh/wms_deploy_key ./scripts/deploy-server.sh
```


## 9. 域名与 HTTPS

已配置域名解析及 ICP 备案：

```text
customprint.icu
www.customprint.icu
```

HTTPS 由容器内 nginx 直接处理（`nginx/default.conf`），证书通过 Let's Encrypt 免费获取。

### 首次申请证书

```bash
dnf install -y certbot
certbot certonly --standalone -d customprint.icu -d www.customprint.icu
```

证书存放在 `/etc/letsencrypt/live/customprint.icu/`，容器通过只读挂载使用。

### 证书自动续期

certbot 安装后自动配置 systemd timer，续期后需 reload 容器 nginx：

```bash
# 添加续期 hook（一次性）
echo "renew_hook = docker exec wms-print-template-nginx nginx -s reload" >> /etc/letsencrypt/renewal/customprint.icu.conf
```

## 10. 生产环境注意事项

- 替换所有 `.env` 中的敏感信息后再部署
- HTTPS 由容器 nginx 承担，证书路径 `/etc/letsencrypt` 以只读挂载
- 数据库切换前执行 `npm run migrate`
- 公开写接口前需添加鉴权
- MySQL 备份需在应用容器外配置
- 容器重启会重新执行全部迁移脚本，迁移 SQL 需保证幂等
