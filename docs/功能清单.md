# 功能清单

本文档列出打印模板中心系统的全部功能，供业务人员了解系统能力。

---

## 1. 登录

| 功能 | 说明 |
|------|------|
| 账号登录 | 输入用户名和密码进入系统 |
| 安全退出 | 点击退出登录，安全退出系统 |

---

## 2. 模板管理

### 2.1 模板列表

| 功能 | 说明 |
|------|------|
| 查看模板 | 查看所有已创建的打印模板，支持翻页浏览 |
| 搜索模板 | 通过模板名称快速找到目标模板 |
| 新建模板 | 创建一个新的空白模板，并进入设计界面 |
| 修改模板 | 进入设计界面修改已有模板 |
| 删除模板 | 删除不再需要的模板 |
| 批量启用 | 一次将多个模板设为可用状态 |
| 批量停用 | 一次将多个模板设为不可用状态 |
| 预览模板 | 查看模板打印出来的实际效果 |
| 复制模板 | 复制一个已有模板，生成内容完全相同的副本 |
| 打印输出 | 选择数据后生成 PDF 文件进行打印 |

### 2.2 模板设计器（核心功能）

模板设计器是用来"画"打印标签的工具，就像用画图软件设计一张卡片。

| 功能 | 说明 |
|------|------|
| 添加文字 | 在模板上放置固定显示的文字 |
| 添加动态文字 | 在模板上放置会自动替换成实际数据的文字，例如显示"库位：A01-001" |
| 添加二维码 | 在模板上放置二维码，扫描后可显示关联信息 |
| 添加条码 | 在模板上放置一维码（类似商品上的条形码） |
| 添加线条 | 在模板上添加装饰性横线或竖线 |
| 添加矩形 | 在模板上添加带背景色的方块 |
| 添加复选框 | 在模板上添加可选中/取消的勾选框 |
| 调整位置 | 用鼠标拖拽调整各元素在模板上的位置 |
| 调整大小 | 拖拽元素边缘的把手改变元素大小 |
| 旋转方向 | 将元素旋转 0°、90°、180° 或 270° |
| 设置样式 | 设置文字大小、加粗、颜色，对齐方式等 |
| 绑定数据 | 将动态元素与具体的数据字段关联 |
| 撤销重做 | 做错操作可以撤销回去，做对了可以重做 |
| 复制粘贴 | 复制已有的元素到其他位置 |
| 保存设计 | 将设计好的模板保存到系统 |
| 预览效果 | 在保存前查看打印出来的实际样子 |
| 放大缩小 | 调整设计界面的显示比例 |

### 2.3 模板启用与停用

| 功能 | 说明 |
|------|------|
| 启用模板 | 将设计完成的模板设为可用，只有启用的模板才能用于打印 |
| 停用模板 | 将模板暂时设为不可用，用于临时关闭某个模板 |

## 3. 字段与数据

### 3.1 模版字段管理

字段是打印模板会用到的数据项目，例如"库位编码"、"商品条码"等。

| 功能 | 说明 |
|------|------|
| 查看字段 | 查看系统中所有业务字段，按模块分类展示 |
| 新建模块 | 创建一个新的业务类型，例如"包材"、"耗材" |
| 修改模块 | 修改业务类型的名称、标签等基本信息 |
| 删除模块 | 删除不需要的业务类型（系统预置的类型不能删） |
| 新增字段 | 为某个业务类型添加新的数据字段，可配置是否启用查询 |
| 修改字段 | 修改已有字段的名称、类型、是否启用查询等属性 |
| 停用字段 | 将字段暂时设为不可用 |
| 启用字段 | 将停用的字段重新设为可用 |
| 删除字段 | 删除不再需要的字段（必填字段不允许删除） |
| 启用查询 | 勾选后该字段在业务数据中支持精确查询；业务编码字段默认启用且不可取消 |

### 3.2 业务数据管理

业务数据是实际要打印的内容，例如具体的库位信息、商品信息。

| 功能 | 说明 |
|------|------|
| 切换业务类型 | 在库位、容器、商品等不同类型之间切换 |
| 查询数据 | 通过业务编码搜索（支持逗号或空格分隔批量查询），同时支持按已启用查询的字段精确筛选 |
| 新增数据 | 添加一条新的业务数据记录 |
| 修改数据 | 修改已有的业务数据内容 |
| 删除数据 | 删除单条不再需要的业务数据 |
| 批量删除 | 一次选中多条数据一起删除 |
| 导入数据 | 通过 Excel 文件批量导入业务数据 |
| 打印数据 | 选中数据后选择模板，生成 PDF 进行打印 |

## 4. 系统管理

### 4.1 用户管理

| 功能 | 说明 |
|------|------|
| 查看用户 | 查看系统中所有使用人员 |
| 新建用户 | 添加一个新的使用账号 |
| 修改用户 | 修改用户的姓名、分配的岗位等 |
| 删除用户 | 删除不需要的账号 |
| 修改密码 | 为用户重置登录密码 |
| 停用账号 | 将账号暂时设为不可登录 |
| 启用账号 | 将停用的账号重新恢复可用 |

### 4.2 角色管理

角色是用来给用户分配权限的，例如"管理员"可以看到所有功能，"操作员"只能看到部分功能。

| 功能 | 说明 |
|------|------|
| 查看角色 | 查看系统中已定义的角色 |
| 新建角色 | 创建一个新的角色，例如"仓库主管" |
| 修改角色 | 修改角色的名称 |
| 删除角色 | 删除不需要的角色 |
| 分配权限 | 给角色勾选允许使用的功能菜单和操作按钮 |

### 4.3 菜单管理

菜单是系统左侧的功能导航树，菜单管理用于维护这个导航结构。

| 功能 | 说明 |
|------|------|
| 查看菜单 | 以树形结构查看所有功能菜单 |
| 新建目录 | 在导航树中添加一个新的分组目录 |
| 新增页面 | 在某个目录下新增一个功能页面 |
| 新增按钮 | 在某个页面下新增一个操作按钮 |
| 修改菜单 | 修改菜单的名称、图标、排序等 |
| 删除菜单 | 删除不需要的菜单项 |

## 5. 权限说明

### 5.1 系统预置账号

| 账号 | 密码 | 权限范围 |
|------|------|----------|
| admin | ------ | 管理员，拥有全部功能权限 |
| business | 123456 | 业务操作员，仅能使用模板和业务数据功能 |

### 5.2 权限管理说明

系统采用"角色+权限"的方式管理操作权限：
- 每个用户属于一个或多个角色
- 每个角色可以访问不同的功能菜单和操作按钮
- 未经授权的功能菜单不会显示在界面中
- 未经授权的按钮会隐藏或显示为不可点击

## 6. 系统预置数据

### 6.1 预置业务模块

系统预先创建了三个常用业务模块，开箱即用：

| 模块 | 用途 | 主键字段 |
|------|------|----------|
| 库位 | 管理仓库中的库位信息 | 库位编码 |
| 容器 | 管理仓库中的容器 | 容器编码 |
| 商品 | 管理商品信息 | 商品编码 |

### 6.2 预置打印模板

系统预置了三个示例模板，可以直接使用或复制后修改：

| 模板名称 | 尺寸 | 状态 |
|----------|------|------|
| 库位标签-100x50 | 100mm × 50mm | 可用 |
| 容器入库标签-100x50 | 100mm × 50mm | 可用 |
| 拣货容器标签-100x150 | 100mm × 150mm | 不可用（停用中） |

### 6.3 预置字段

各业务模块预置的字段如下：

**库位模块**：库位编码、库位前缀、排、列、层、方向标、区域仓编码、物理仓编码

**容器模块**：容器编码、区域仓编码、物理仓编码

**商品模块**：商品编码、商品条码、客户商品编码
