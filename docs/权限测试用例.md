# 权限系统测试方案

## 测试环境
- 前端: http://127.0.0.1:5173
- 账号: admin/123456 (管理员), business/123456 (业务)
> 完整预置账号信息见 [功能清单](功能清单.md#5-权限说明)

## 测试用例

### TC1: 登录 — admin
```bash
goto http://127.0.0.1:5173/login
fill e21 "admin"
fill e34 "123456"
click e45
# 预期: 跳转 /templates, 侧边栏显示 3 个菜单组
eval "() => window.location.href" → "/templates"
eval "() => document.querySelectorAll('.ant-menu-item-group').length" → 3
```

### TC2: 登录 — business
```bash
# 先登出, 清除 session
goto http://127.0.0.1:5173/login
fill e21 "business"
fill e34 "123456"
click e45
# 预期: 跳转 /templates, 侧边栏显示 2 个菜单组(无系统管理)
eval "() => window.location.href" → "/templates"
eval "() => document.querySelectorAll('.ant-menu-item-group').length" → 2
```

### TC3: business 页面权限 — 字段字典
```bash
# 登录 business 后
goto http://127.0.0.1:5173/fields
# 预期: 0 console errors, 按钮仅显示 编辑/停用 (无删除)
console → 0 errors
eval "() => { const btns = document.querySelectorAll('.ant-table-row')[0]?.querySelectorAll('button'); return Array.from(btns||[]).map(b=>b.textContent.trim()).join(','); }" → "编辑,停用"
```

### TC4: business 页面权限 — 业务数据
```bash
goto http://127.0.0.1:5173/business
# 预期: 新增/导入/打印/删除按钮可见
eval "() => Array.from(document.querySelectorAll('button')).filter(b=>b.offsetParent).map(b=>b.textContent.trim()).filter(t=>t).join(',')" → 包含 "新增" 和 "导入"
```

### TC5: business 无权访问系统管理
```bash
goto http://127.0.0.1:5173/system/users
# 预期: 跳转 /403
eval "() => window.location.href" → "/403"
```

### TC6: admin 全部权限
```bash
# 登录 admin
goto http://127.0.0.1:5173/system/users
# 预期: 正常加载, 表格有用户数据
eval "() => document.querySelectorAll('.ant-table-row').length" → ≥1
```

### TC7: 角色管理 — 权限卡片
```bash
# 登录 admin
goto http://127.0.0.1:5173/system/roles
# 点击编辑
eval "() => { const b=document.querySelectorAll('.ant-table-row button'); for(const btn of b){if(btn.textContent.includes('编')){btn.click();break;}} }"
# 预期: 弹窗中显示 6 个 checkbox(页面级) + 30+ tag(按钮级)
eval "() => document.querySelector('.ant-modal')?.querySelectorAll('.ant-checkbox').length" → 6
```

### TC8: 空目录隐藏
```bash
# 创建测试角色: 只有 template:view
# 登录该角色
# 预期: 侧边栏仅显示"模板管理", 无"字段与数据"和"系统管理"
eval "() => document.querySelectorAll('.ant-menu-item-group').length" → 1
```

### TC9: 菜单管理树
```bash
# 登录 admin
goto http://127.0.0.1:5173/system/menus
# 预期: 左侧树展开全部, 至少 30 个节点
eval "() => document.querySelectorAll('.ant-tree-title').length" → ≥30
```

### TC10: 权限即时生效
```bash
# 登录 admin, 给 business 角色移除 field:view 权限
# business 用户强制刷新 /fields
# 预期: 跳转 403
eval "() => window.location.href" → "/403"
```
