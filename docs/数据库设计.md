# 数据库设计

本文档说明 打印模板中心的数据库结构。

## 数据库信息

```text
Database: wms_print_template
```

迁移脚本目录：

```text
backend/migrations/
```

## 数据表

| 表名 | 说明 |
|------|------|
| print_template | 打印模板主表 |
| print_template_element | 模板元素（组件） |
| print_field_dict | 字段字典 |
| print_business_module | 业务模块（模板类型元数据） |
| print_log | 打印日志 |
| operation_log | 操作日志 |
| product | 商品业务数据 |
| container | 容器业务数据 |
| location | 库位业务数据 |

## 种子数据

迁移后自动写入：

```text
字段字典：
- LOCATION   库位字段
- CONTAINER  容器字段
- PRODUCT    商品字段

仓库：
- JP-TYO-01（东京）
- US-LAX-01（洛杉矶）
- DE-FRA-01（法兰克福）
```

## 字段字典定义

### 库位（LOCATION）

| 字段编码 | 名称 | 数据库列 | 必填 |
|---------|------|----------|------|
| locationCode | 库位编码 | location_code | 是 |
| locationPrefix | 库位前缀 | location_prefix | — |
| row | 排 | location_row | — |
| column | 列 | location_column | — |
| level | 层 | location_floor | — |
| directionMark | 方向标 | direction_flag | — |
| warehouseCode | 区域仓编码 | region_warehouse_code | — |
| areaCode | 物理仓编码 | physics_warehouse_code | — |

### 容器（CONTAINER）

| 字段编码 | 名称 | 数据库列 | 必填 |
|---------|------|----------|------|
| containerCode | 容器编码 | container_code | 是 |
| warehouseCode | 区域仓编码 | region_warehouse_code | — |
| areaCode | 物理仓编码 | physics_warehouse_code | — |

### 商品（PRODUCT）

| 字段编码 | 名称 | 数据库列 | 必填 |
|---------|------|----------|------|
| productCode | 商品编码 | sku | 是 |
| customerProductCode | 客户商品编码 | customer_code | — |

### 方向标规则

```
direction_flag = 1  → 显示"向上" → 打印 ↑
direction_flag = 2  → 显示"向下" → 打印 ↓
direction_flag = 空 → 显示空 → 打印空
```

## 存储说明

模板元素中未单独建列的字段存储在 `extra_json` 字段中。后端从 `print_template`、`print_template_element` 两张表还原前端 DSL。
