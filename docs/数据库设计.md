# 数据库设计

本文档说明 WMS 打印模板中心的数据库结构。

## 数据库信息

```text
Database: wms_print_template
```

迁移脚本目录：

```text
backend/migrations/
```

## 数据表

| 表名 | 说明 |
|------|------|
| print_template | 打印模板主表 |
| print_template_element | 模板元素（组件） |
| print_template_warehouse | 模板关联仓库 |
| print_field_dict | 字段字典 |
| print_log | 打印日志 |
| operation_log | 操作日志 |
| product | 商品业务数据（本地） |
| container | 容器业务数据（本地） |
| location | 库位业务数据（本地） |

## 种子数据

迁移后自动写入：

```text
模板：
- TPL_LOCATION_10050    库位标签
- TPL_CONTAINER_IN_10050  容器标签
- TPL_PICKING_100150    拣货标签

字段字典：
- LOCATION   库位字段
- CONTAINER  容器字段
- PRODUCT    商品字段

仓库：
- JP-TYO-01
- US-LAX-01
- DE-FRA-01
```

## 字段字典定义

### 库位（LOCATION）

| 字段编码 | 名称 | 必填 |
|---------|------|------|
| warehouseCode | 区域仓编码 | 是 |
| areaCode | 物理仓编码 | 否 |

### 容器（CONTAINER）

| 字段编码 | 名称 | 必填 |
|---------|------|------|
| containerCode | 容器编码 | 是 |
| warehouseCode | 区域仓编码 | 否 |
| areaCode | 物理仓编码 | 否 |

### 商品（PRODUCT）

| 字段编码 | 名称 | 必填 |
|---------|------|------|
| productCode | 商品编码 | 是 |
| customerProductCode | 客户商品编码 | 否 |

## 存储说明

模板元素中未单独建列的字段存储在 `extra_json` 字段中。后端从 `print_template`、`print_template_element`、`print_template_warehouse` 三张表还原前端 DSL。

## 迁移脚本

```text
001_init.sql                              初始化表结构和种子数据
002_update_field_dict.sql                 更新字段字典
003_add_product_field_dict.sql            新增商品字段字典
004_add_business_data.sql                 新增业务数据表
005_add_template_print_rotation.sql       新增模板打印旋转
006_normalize_template_status.sql         规范化模板状态
007_prune_unused_schema.sql               清理无用字段
008_update_container_warehouse_optional.sql  容器区域仓改为可选
009_add_field_preview_values.sql          新增字段预览值
010_remove_container_purpose_usagescene.sql  移除容器用途和使用场景
011_add_local_business_tables.sql         新增本地业务数据表
012_make_region_warehouse_optional.sql    区域仓改为可选
```
