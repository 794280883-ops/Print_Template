# 环境变量说明

本文档列出项目中所有环境变量文件及其字段含义。

## 文件概览

| 文件 | 用途 | 是否提交Git |
|------|------|-------------|
| `.env.example` | 本地开发模板 | ✅ 提交 |
| `.env.production.example` | 生产部署模板 | ✅ 提交 |
| `.env.test.example` | 测试环境模板 | ✅ 提交 |
| `.env` | 本地实际配置 | ❌ gitignore |
| `.env.production` | 生产实际配置 | ❌ gitignore |

---

## 1. `.env.example` — 本地开发

用于 `./scripts/dev-local.sh` 启动本地开发环境。

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `MYSQL_ROOT_PASSWORD` | `wms_root_password` | MySQL root 密码 |
| `MYSQL_DATABASE` | `wms_print_template` | 数据库名 |
| `MYSQL_USER` | `wms_app` | 应用数据库用户名 |
| `MYSQL_PASSWORD` | `wms_app_password` | 应用数据库密码 |
| `BACKEND_PORT` | `3001` | 后端端口 |
| `FRONTEND_PORT` | `5173` | 前端端口 |
| `HTTP_PORT` | `8080` | Docker HTTP端口 |
| `CORS_ORIGIN` | `http://127.0.0.1:8080` | 跨域允许来源 |
| `VITE_API_BASE_URL` | `/api/v1` | 前端API基础路径 |
| `TEST_DB_HOST` | `127.0.0.1` | 外部测试数据库地址 |
| `TEST_DB_PORT` | `3306` | 外部测试数据库端口 |
| `TEST_DB_USER` | `readonly_user` | 外部测试数据库用户 |
| `TEST_DB_PASSWORD` | `change_me` | 外部测试数据库密码 |
| `TEST_DB_NAME` | `fbu_wms_base_de` | 外部测试数据库名 |
| `TEST_DB_CONNECTION_LIMIT` | `5` | 测试连接池上限 |

### 使用方式

```bash
cp .env.example .env
# 编辑 .env 填写本地 MySQL 密码
./scripts/dev-local.sh
```

---

## 2. `.env.production.example` — 生产部署

用于服务器 `docker compose up` 部署。

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `JWT_SECRET` | （需替换） | JWT签名密钥，至少32字符 |
| `MYSQL_ROOT_PASSWORD` | （需替换） | MySQL root 强密码 |
| `MYSQL_DATABASE` | `wms_print_template` | 数据库名 |
| `MYSQL_USER` | `wms_app` | 应用数据库用户名 |
| `MYSQL_PASSWORD` | （需替换） | 应用数据库强密码 |
| `HTTP_PORT` | `80` | HTTP入口端口 |
| `CORS_ORIGIN` | `https://customprint.icu` | 生产域名 |
| `VITE_API_BASE_URL` | `/api/v1` | 前端API基础路径 |

### 使用方式

```bash
cp .env.production.example .env
nano .env  # 替换所有敏感信息
docker compose up -d --build
```

---

## 3. `.env.test.example` — 测试环境

用于连接外部 WMS 数据库进行只读测试。

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `PORT` | `3001` | 测试服务端口 |
| `NODE_ENV` | `test` | 运行环境标识 |
| `CORS_ORIGIN` | `http://127.0.0.1:5173` | 跨域来源 |
| `TEST_DB_HOST` | （需填写） | 外部测试库地址 |
| `TEST_DB_PORT` | `3306` | 外部测试库端口 |
| `TEST_DB_USER` | `readonly_user` | 只读用户 |
| `TEST_DB_PASSWORD` | （需填写） | 测试库密码 |
| `TEST_DB_NAME` | `fbu_wms_base_de` | 测试库名 |
| `TEST_DB_CONNECTION_LIMIT` | `5` | 连接池上限 |
| `VITE_API_BASE_URL` | `/api/v1` | 前端API基础路径 |

### 使用方式

```bash
cp .env.test.example .env.test
# 编辑 .env.test 填写真实测试库连接信息
npm --prefix backend run test
```
